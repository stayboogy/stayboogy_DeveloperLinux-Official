# After Install from Debian Minimal ISO

```
git clone https://github.com/stayboogy/DeveloperLinux-Official.git
cd DeveloperLinux-Official
git checkout arm64

sudo chmod 7777 ./install
sudo chmod 7777 ./spkgi
sudo ./spkgi

enjoy
```
